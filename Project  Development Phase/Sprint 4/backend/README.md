# Expenso Bakcend - flask

### create virtual env

``` python -m venv venv ```

### activate venv

``` .\venv\Scripts\activate ```
### api keys
  1. Add your db endpoint
  2. Set secret 
  3. Set sendgrid api key 
### install requirements

``` pip install -r requirements.txt ```

### run app

``` python app.py ```
 
### app is running on localhost:5000/
